# Lista pomysłów na bliższe i dalsze inicjatywy

## Cel
- Zbiór inicjatyw/pomysłów, które **nie są jeszcze zadaniami**.
- Jeśli pomysł ma sens, a nie chcesz go zgubić — dopisz go tutaj.

## Zasady
- `Created` zawsze co do minuty (`Europe/Warsaw`). Jeśli nie znasz godziny (np. migracja starych wpisów) użyj `00:00`.
- `Status` opisuje stan pomysłu (np. `todo`, `in-progress`, `done`, `on-hold`).
- Checkbox `[ ]` / `[x]` traktuj jako prosty “done/not done”.
- Gdy pomysł zamieniasz w task: dopisz `Task: <ID-Z>_<status>` w treści wpisu.

## Format wpisu (szablon)

```markdown
- [ ] [P?] Tytuł idei
  - Created: YYYY-MM-DD HH:MM (Europe/Warsaw)
  - Status: todo | in-progress | done | on-hold
  - Cel/Wartość: ...
  - Zakres (in/out): ...
  - Zależności: ...
  - Ryzyka: ...
  - Mierniki sukcesu: ...
  - Owner: ...
  - Horyzont: ...
  - Task: <ID-Z>_<status> (opcjonalnie)
```
