# additional-notes - szablon

Dodawaj pliki `<ID-T>.md` (np. `01.md`) gdy notatki nie mieszczą się w `tasks.md`.

Minimalny wzorzec jednego pliku:

```markdown
# <ID-T> - Krótki tytuł

## Surowe notatki / logi (lokalne)
...

## Wnioski dla kolejnych kroków
[HANDOFF: <lista ID-T, np. 02>]
- ...

## Powiązane zadania
- Kolejne: ...
- Ewentualne: ...
```
