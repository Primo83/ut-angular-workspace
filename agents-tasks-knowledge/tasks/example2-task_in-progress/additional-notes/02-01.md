# 02-01 – Endpoint `GET /availability`

## Surowe notatki / logi (lokalne)

- Mapowanie kalendarza → sloty godzinowe:
  - Brak wpisu = `off`.
  - Wpis całodniowy = `off` z komentarzem (np. urlop).
  - Wpis częściowy = `partial` z zakresem godzin (np. `09:00-13:00`).
